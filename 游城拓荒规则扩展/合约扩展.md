# 定义阐释
## 合约
合约是本扩展的核心组分，是一份具有行为约束力的文书
### 结构
1. 合约标题：简要概括合约内容方便查找
2. 合约内容
	1. 合约条款：规定合约各方应当履行的义务
	2. 违约赔偿：规定违约方应对其他合约签署方给予的赔偿
3. 有效时限：以回合为单位，规定合约的有效时限
4. 合约签订方署名
